# 认识音符：Do Re Mi

在西方音乐中，基础音阶由 **七个音符** 组成，对应钢琴上的七个白键。

我们用 **唱名法（Solfège）** 来称呼它们：

| 唱名 | 音名 | 说明 |
|------|------|------|
| **Do** | C | 音阶的起点 |
| **Re** | D | |
| **Mi** | E | |
| **Fa** | F | |
| **Sol** | G | |
| **La** | A | |
| **Si** | B | 音阶的终点 |

## 小技巧

> 🎵 记忆唱名最有效的方式是**边听边唱**。试着点击琴键的同时，跟着唱出 Do、Re、Mi……
